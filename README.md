## ⚡ Features

*   🔄 **Universal Converter**: Convert between PDF and Word (.docx), PowerPoint (.pptx), Excel (.xlsx), Text, HTML, and Images. Create PDFs from Images.
*   💧 **Deep Clean Engine**: Advanced watermark removal. Strip annotations, vector graphics, repeating text, or perform an extreme text purge.
*   ✂️ **Auto-Crop Engine**: Automatically detect and slice away blank white boundaries or use manual matrix margins.
*   📚 **Merge & Split**: Combine multiple PDFs into a single document or extract specific pages using range expressions (e.g., `1-3, 5, 8-10`).
*   🗜️ **Ultimate Compressor**: Reduce file sizes massively via structural deflation, garbage collection, and aggressive image downscaling.
*   🎨 **Dark Mode Inverter**: Convert PDF pages into inverted high-contrast dark-mode images for easier reading at night.
*   🖼️ **Neural Background Eraser**: Isolate foreground elements using U2Net deep learning AI (`rembg`).
*   🔐 **Security Protocols**: Encrypt your PDF with military-grade AES-256 or unlock secured documents.

## 🛠️ Installation

PDF Studio Pro runs entirely locally on your machine, ensuring complete data privacy.

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/SONUVERMA11/PDF_Master_Pro.git
    cd PDF_Master_Pro
    ```

2.  **Execute the setup script:**
    ```bash
    chmod +x run.sh
    ./run.sh
    ```
    *(The script will automatically create a virtual environment, install all dependencies, and launch the application.)*

## 🚀 Usage

Once the server is running, open your web browser and navigate to:
```text
http://localhost:8501
```

> **Note:** The first time you use the AI Background Remover, it will briefly pause to download the required neural network weights (`u2net`).
